Web Accessibility

  • meta ALT tags: I add alt tags to images to make my site more accessible. This helps those who have visual impairments. Also this protects against broken links.

  • Mobile Devices: To make my site easier to view on smaller devices I added a setting to my css file in order to make the page resize on smaller devices such as mobile phones.

  • Site Style: The site's styling I wanted to keep clean by adding colors that would show up clear on background. All the styling is done in separate css files to keep things modular.

Web Usability

  • Site Content: I tired to model the site's content such that content is where one would expect to find it. Also gave site content appropriate header/titles to give the user an idea of the following content.

  • Site navigation: To make my site easier to use and navigate I added a navigation bar which is fixed to the top of the web browser when scrolling to different sections of a page. The navigation also highlights the active page in the navigation bar so user can easily find the current page.

  • Site Load Time: In order to make sure my users don't feel that the site is non-responsive I made sure my site load time is a reasonable length of time. The images I used were jpeg format to reduce load time.
